# Quantitative-Finance-Regression-Analysis
Python and Excel-based quantitative finance analysis using hedge fund returns, S&amp;P 500 factors, and OLS regression.

## Project Overview
This project applies quantitative finance methods to analyze hedge fund return data using Python and Excel. The analysis uses OLS regression to examine the relationship between hedge fund strategy returns and S&P 500 market movement factors.

## Tools Used
- Python
- pandas
- statsmodels
- Excel
- OLS Regression
- Quantitative Finance Analysis

## Files Included
- `DIVYA 05_11_2025.ipynb - Colab.pdf`: Python/Colab output showing regression code and results
- `QuantHWreplication-DA.xlsx`: Excel file used for quantitative finance replication analysis

## Analysis Summary
The Python analysis imports hedge fund and S&P 500 factor data, runs an OLS regression model, and reviews key regression outputs such as coefficient estimates, p-values, R-squared, adjusted R-squared, and model significance.

## Key Skills Demonstrated
- Used Python to perform financial data analysis
- Applied OLS regression using statsmodels
- Interpreted regression output for quantitative finance analysis
- Worked with Excel-based financial datasets
- Connected market factor variables with hedge fund return behavior
